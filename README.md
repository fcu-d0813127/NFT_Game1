## 功能
- 使用方向鍵左右來移動玩家方塊(正方形)
- 綠色背景為場景一，持續往左碰到邊屆時會進入到場景二
- 場景二會出現敵人(菱形)，背景顏色改為紅色
- 玩家位置會從畫面右手方繼續移動
- 場景二繼續往左時，將不會切換場景，往又碰到邊界時會回到場景一
- 玩家位置會從畫面左手方繼續移動
- 按鍵K會使敵人消失
